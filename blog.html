How K-Means Works (Intuitively)

K-Means is a clustering algorithm used to group similar data points together. It works by selecting a number of clusters (K) and randomly initializing centroids.

The algorithm assigns each data point to the nearest centroid and recalculates the centroids based on the average of points in each cluster until stable.

It is widely used in customer segmentation and pattern recognition.
